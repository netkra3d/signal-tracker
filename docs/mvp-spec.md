# Signal Tracker MVP

## 목표
- 240분봉 데이터 조회
- EMA / RSI / MACD 기반 신호 생성
- 분할매수 / 분할매도 기록
- 일간 / 월간 / 연간 수익 분석

## 현재 포함 범위
- Next.js App Router UI
- localStorage 기반 거래 저장
- 업비트 240분봉 실데이터 연동
- 미국 ETF / 국내 ETF 실가격 provider 경로 추가
- 대시보드 / 자산 / 거래 / 분석 / 설정 화면
- JSON export / import 백업 기능

## 추후 확장
- localStorage 저장소를 DB 저장소로 교체
- 텔레그램 / 웹 푸시 알림
- 백테스트
- 자동매매 API 연결
